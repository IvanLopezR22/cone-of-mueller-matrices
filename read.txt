INSTALACIÓN:
Una vez dentro de la carpeta del pryecto, ejecutar este comando
pip install -r requirements.txt
(Istalar las librerías en las versiones que usamos)

EJECUCIÓN
Para ejecutarse se tienen dos opciones 
1. Sin herramientas de desarrollo
1.1. Acceder a carpeta dist/index y ejecutar el archivo index.exe (windows)
2. Con herramientas de desarrollo.
2.1 Ejecutar index.py como punto de entrada. 
(entry point)
